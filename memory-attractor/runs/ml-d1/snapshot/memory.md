## Story
I was staring at the definition of a "balanced" number: $n + r(n) = x^2$ and $n - r(n) = y^2$. This implies $2n = x^2 + y^2$ and $2r(n) = x^2 - y^2$, requiring $x, y$ to have the same parity. If $y=0$, $n$ is a palindrome and $n = x^2/2 = 2k^2$. Thus, any palindrome that is twice a square is balanced.

For single digits, $n-r(n)=0$ is always a square. $2n=x^2$ requires $n$ to be twice a square. Only 2 and 8 worked.

For two digits, $n=10a+b$. $n+r(n)=11(a+b)$ and $n-r(n)=9(a-b)$. For the sum to be a square, $a+b=11$. For the difference to be a square, $a-b$ must be a square. Solving $a+b=11$ and $a-b=k^2$ with digit constraints yielded only $k^2=1$, giving $a=6, b=5$. So 65 is the unique two-digit balanced number. It is not a palindrome ($y=3$).

For three digits, $n=100a+10b+c$. $n-r(n)=99(a-c)$. For this to be a square, $11(a-c)$ must be a square. Since $|a-c| \le 9$, we must have $a=c$. Thus, all three-digit balanced numbers are palindromes. The condition reduces to $n+r(n)=2n=x^2$, so $n=2k^2$. We need a palindrome of the form $2k^2$. Testing $a=2$ (since $a \equiv 2k^2 \pmod{10}$ implies $a \in \{2,8\}$), we found $n=242$ works ($242+242=484=22^2$). $a=8$ yielded no solutions. So 242 is the unique three-digit balanced number.

For four digits, palindromes $abba$ require $2(1001a+110b)$ to be a square, implying $91a+10b = 22m^2$. No digit solutions exist. Non-palindromic four-digit cases are complex, but the pattern suggests scarcity.

The known balanced numbers are 2, 8, 65, 242. The core question remains: Are there infinitely many? This reduces to whether there are infinitely many palindromes of the form $2k^2$ (since non-palindromic solutions seem rare, e.g., only 65 for 2 digits). This is likely an open problem in number theory regarding palindromes in quadratic sequences. I suspect the answer is yes, but constructing an infinite family eludes me. The algebraic structure is clear, but the digit-reversal constraint makes it combinatorially difficult. I will leave the search for larger palindromic squares of the form $2k^2$ as an open thread.
